# WIT
### *wait — you're not ABC?*

> You are not learning English. You are learning power.

---

## 什么是 WIT？

你每天开英文会议、做 pitch、跟老外 1-on-1。

你说的不是错的。但听起来，就是差点意思。

**流利**和**听起来聪明**，是两件完全不同的事。

WIT 帮你做后者。

---

## 它能做什么

**🎙 录音 + 实时转写**
直接说，不用粘贴。说完点分析。

**🔍 AI 深度分析**
不只纠错——找出你的表达和 a16z、YC 那种说话方式差在哪。
每个问题都告诉你：原句 → 更好的说法 → 为什么这样更地道。

**✍️ 改写全文**
同样的意思，AI 给你一版真正自然的英文。对比着看，差距一目了然。

**📚 Problem Library**
每次练习的问题自动入库。练多了，你的高频盲点自己浮出来。
出现 2 次以上？标红。这就是你最需要攻克的地方。

**💡 Vocab List**
把你够不到的词提前送到你眼前。
支持 bulk import——一次扔 100 个词，有中文保留，没有自动补，全自动。

**🔊 发音 + 中文释义**
每个词一键听发音，一键看中文。学了就能用。
